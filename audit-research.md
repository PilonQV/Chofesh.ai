# Security & Audit Tools Research

## Security Audit Tools (2025)

### Static Application Security Testing (SAST)
1. **SonarQube** - Broad static analysis platform combining SAST with code quality checks
2. **Snyk** - Fast vulnerability detection for npm packages
3. **Semgrep** - Customizable static analysis with control
4. **Checkmarx** - Highly accurate static code analysis
5. **ESLint Security Plugin** - JavaScript/TypeScript security linting

### Dynamic Application Security Testing (DAST)
1. **OWASP ZAP** - Free, open-source penetration testing
2. **Burp Suite** - Professional web security testing
3. **Wapiti** - Web application vulnerability scanner

### Dependency Scanning
1. **npm audit** - Built-in npm vulnerability scanning
2. **Snyk** - Comprehensive dependency vulnerability detection
3. **OWASP Dependency-Check** - Open-source SCA tool

### Tools We'll Use for This Audit
- npm audit (dependency vulnerabilities)
- ESLint with security plugins (code analysis)
- Manual OWASP Top 10 review
- Custom security checks for auth/API

## UI/UX Audit Tools

### Performance & Quality
1. **Google Lighthouse** - Performance, accessibility, SEO, best practices
2. **GTmetrix** - Detailed performance reports
3. **WebPageTest** - In-depth performance analysis

### Accessibility Testing
1. **WAVE** - Web accessibility evaluation
2. **axe DevTools** - Automated accessibility testing
3. **AccessibilityChecker.org** - Free ADA/WCAG compliance scan

### User Experience Analysis
1. **Hotjar** - Heatmaps and session recordings
2. **Microsoft Clarity** - AI-powered behavior analysis
3. **Mixpanel** - Product analytics

### Tools We'll Use
- Lighthouse (performance, accessibility, SEO)
- Manual WCAG review
- Mobile responsiveness testing
- User flow analysis
